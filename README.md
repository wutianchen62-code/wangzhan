# AI动态海报生成器

🎨 一个基于AI的交互式动态海报生成器，用户可以输入文本内容并选择动态效果，系统会自动生成精美的动态海报。

## 功能特点

### ✨ 核心功能
- **智能海报生成**：用户输入标题和内容，AI自动生成动态海报
- **多种动画效果**：支持脉冲、弹跳、淡入淡出、滑动、旋转等效果
- **丰富色彩主题**：提供多种渐变色彩方案
- **实时预览**：输入内容时实时预览效果
- **响应式设计**：适配各种屏幕尺寸

### 🚀 技术特色
- **前后端分离**：前端使用纯HTML/CSS/JavaScript，后端使用Flask
- **AI增强**：后端提供内容增强和智能建议
- **本地运行**：无需网络连接，完全本地化运行
- **模块化设计**：易于扩展和维护

## 快速开始

### 方法一：直接运行HTML（简单）
1. 双击 `index.html` 文件
2. 在浏览器中打开即可使用

### 方法二：使用Flask后端（推荐）
1. 安装Python依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 启动Flask服务器：
   ```bash
   python app.py
   ```

3. 打开浏览器访问：
   ```
   http://localhost:5000
   ```

## 使用说明

### 1. 输入内容
- **标题**：输入海报的主要标题
- **内容**：输入海报的详细内容
- **动态效果**：选择喜欢的动画效果
- **色彩主题**：选择配色方案

### 2. 生成海报
- 点击"🎯 生成动态海报"按钮
- 系统会调用AI生成器创建海报
- 生成的海报会显示在右侧预览区域

### 3. 实时预览
- 输入内容时，系统会实时更新预览
- 可以随时调整参数查看效果

## 技术架构

### 前端技术栈
- **HTML5**：语义化标签和现代结构
- **CSS3**：Flexbox布局、Grid布局、动画效果
- **JavaScript ES6+**：面向对象编程、异步处理

### 后端技术栈
- **Flask**：轻量级Web框架
- **Python**：后端逻辑处理
- **JSON API**：前后端数据交互

### AI功能
- **内容增强**：自动优化和丰富用户输入
- **智能建议**：提供标题和内容的建议
- **动态生成**：根据参数生成定制化海报

## 自定义开发

### 添加新的动画效果
1. 在 `style.css` 中添加新的 `@keyframes` 动画
2. 在 `script.js` 中更新动画选择器
3. 在 `app.py` 中更新动画生成逻辑

### 添加新的色彩主题
1. 在 `style.css` 中添加新的渐变样式
2. 在 `script.js` 中更新颜色选择器
3. 在 `app.py` 中更新颜色生成逻辑

### 扩展AI功能
1. 在 `app.py` 的 `PosterGenerator` 类中添加新方法
2. 创建新的API端点
3. 在前端添加对应的交互功能

## 故障排除

### 常见问题

**Q: 海报生成失败怎么办？**
A: 检查浏览器控制台是否有错误信息，确保后端服务正常运行。

**Q: 动画效果不显示？**
A: 检查浏览器是否支持CSS3动画，尝试刷新页面。

**Q: 如何导出海报？**
A: 目前支持屏幕截图功能，后续版本将添加导出功能。

### 开发调试
- 打开浏览器开发者工具查看控制台输出
- 检查网络请求是否正常
- 验证API端点是否正确响应
